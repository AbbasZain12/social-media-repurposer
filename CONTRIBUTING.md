# Contributing to Social Media Repurposer

Thank you for your interest in contributing to this project.

## How to Contribute

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes and commit them: `git commit -m "Add your feature"`
4. Push to your branch: `git push origin feature/your-feature-name`
5. Open a Pull Request with a clear description of the changes.

## Guidelines

- Follow the existing code structure and naming conventions.
- Write clear, descriptive commit messages.
- Add comments for any complex logic.
- Test your changes before submitting a pull request.

## Reporting Issues

Use GitHub Issues to report bugs or suggest new features. Please include steps to reproduce the issue and any relevant error messages.
