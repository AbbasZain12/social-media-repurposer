# Social Media Repurposer (Frontend)

React + Vite + Tailwind CSS frontend for generating repurposed social media content from YouTube videos.

## Features

- User login and signup
- JWT-based authentication with protected routes
- YouTube video data fetching
- AI content generation via Google Gemini
- Generation history page
- Responsive design

## Tech Stack

React 18, Vite, Tailwind CSS, React Router DOM, Axios

## Getting Started

### Prerequisites

- Node.js v16 or higher
- Backend server running at `http://localhost:5000`

### Installation

```bash
npm install
```

Create a `.env` file in the project root:

```env
VITE_API_URL=http://localhost:5000
```

Start the development server:

```bash
npm run dev
```

Frontend runs at `http://localhost:5173`.
